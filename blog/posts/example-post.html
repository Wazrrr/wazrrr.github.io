Hello, world: building this website

Oct 20, 2025 · Ziren Wang

This is a sample post to show the blog structure. Posts live in /blog/ and share a common header, footer, and stylesheet.

You can write with plain HTML, or integrate a static site generator later if you want Markdown and automatic indexing.

Tips:

  • Keep titles concise and add a <meta name="description">.
  • Use relative links like /assets/... so everything works on most static hosts.
  • Add an rss.xml later if you want subscribers.
